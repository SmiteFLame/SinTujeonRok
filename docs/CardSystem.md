## 1. 패 구조 개요

### 패 = 패 종류 × 이름(개체) × 품계

```
팔목(8종류) × 지역별 패 × 품계(24단) = 풍부한 수집/성장 콘텐츠
```

### 핵심 변경점 (v0.1 → v0.2)
| 항목 | v0.1 | v0.2 |
|------|------|------|
| 등급 체계 | 4등급 (범/수/걸/신) | **24품계** (지역별 3품씩) |
| 연마 효과 | 장착 패만 적용 | **장착 +100, 기본 공력(功力) +1** |
| 지역 연동 | 없음 | **지역별 인연(因緣) & 해금 조건** |
| 최종 패 | 누구나 획득 | **조건부 해금** (여러 패 연마 필요) |

## 2. 품계(品階) 시스템 (24단)

### 등급 → 품계 개편
기존 4등급 대신, 조선시대 **관직 품계**에서 영감을 받은 24단 시스템.
8개 지역 × 3품계씩 = 24품계.

## 3. 연마(鍊磨) 시스템 (공력 연동)

### 핵심 원칙: "연마는 절대 버려지지 않는다"

패를 연마하면 두 가지 효과가 동시에 적용됩니다:

```
┌───────────────────────────────────────┐
│  패 연마 +1 시                        │
│                                       │
│  ① 해당 패 전투력: +100 (직접 효과)   │
│  ② 패사 기본 공력:  +1  (영구 효과)   │
│                                       │
│  → 패를 바꿔도 공력은 남아있다!       │
└───────────────────────────────────────┘
```

### 연마 상세

#### 장착 효과 (해당 패에만 적용)
```
연마 +1당 전투 스탯 증가:
  공(攻):   +100
  수(守):   +50
  기력(氣力): +200
  신(速):   +10
```

#### 공력(功力) - 영구 효과
```
연마 +1당 패사 기본 스탯 증가:
  기본 공(攻):   +1
  기본 수(守):   +0.5
  기본 기력:     +2
  기본 신(速):   +0.1

※ 모든 패에 적용되는 영구 스탯
※ 한 번 올린 공력은 절대 사라지지 않음
```

#### 연마 비용 & 성공률
```
연마 단계: +0 ~ +30

구간 1 (안전 구간): +0 ~ +10
  비용: 은냥 100 ~ 5,000 (점진 증가)
  성공률: 100%

구간 2 (도전 구간): +11 ~ +20
  비용: 은냥 5,000 ~ 50,000
  성공률: 90% → 50% (점진 감소)
  실패 시: 연마 단계 유지, 재료만 소모

구간 3 (극한 구간): +21 ~ +30
  비용: 은냥 50,000 ~ 500,000
  성공률: 40% → 10%
  실패 시: 연마 단계 유지, 재료만 소모
```

#### 공력 누적 예시
```
시나리오: 장사(人) +10, 포졸(人) +5, 참새(鳥) +8 연마한 경우

총 공력 = (10 + 5 + 8) × 1 = 23 공력

→ 모든 패에 적용되는 영구 스탯:
  기본 공(攻):   +23
  기본 수(守):   +11.5
  기본 기력:     +46
  기본 신(速):   +2.3

→ 새로운 패를 개봉해도 이 공력은 이미 적용된 상태!
→ 연마를 많이 할수록 어떤 패든 강해짐
```

## 4. 지역별 패 & 신물(神物)

### 구조: 일반 패 6장 + 신물(神物) 2장

각 지역에서 8종 패 중 **6장은 일반 획득**, **2장은 🔱신물(조건부 혼합 연마 해금)**.
신물은 해당 지역과 이전 지역의 다양한 패를 골고루 연마해야 해금됩니다.

```
지역당 구성:
├── 일반 패 6장: 스토리/방치/두목/관문으로 획득
└── 🔱 신물(神物) 2장: 조건부 혼합 연마로 해금
```

## 8. 패 스탯 구조 (데이터 모델)

```
CardData {
    id: string              // 고유 ID (예: "HY_IN_001")
    name: string            // 패 이름 (예: "장사")
    type: PaeType           // 패 종류 (人/魚/鳥/雉/星/馬/獐/兎)
    region: Region          // 소속 지역
    gradeRange: int[2]      // 품계 범위 [min, max] (예: [1, 3])
    currentGrade: int       // 현재 품계 (1~24)
    enhanceLevel: int       // 연마 단계 (0~30)

    // 기본 패결 스탯
    baseAttack: int         // 공(攻)
    baseDefense: int        // 수(守)
    baseHp: int             // 기력(氣力)
    baseSpeed: int          // 신(速)

    // 술(術)/기운(氣韻)
    skill: SkillData                   // 술(術)
    passiveAbility: PassiveData[]      // 기운(氣韻) - 6품, 12품, 18품, 24품 해금
    ultimateSkill: SkillData           // 비술(秘術) - 18품 이상 해금

    // 해금 조건
    unlockCondition: UnlockCondition   // null이면 즉시 획득 가능
}

UnlockCondition {
    requiredCards: {cardId: string, minEnhance: int}[]
    requiredGongryeok: int          // 필요 공력
}
```

## 10. 개봉(開封) 시스템

### 개봉 확률 (품계 기반)
| 개봉 결과 | 일반 개봉 | 고급 개봉 |
|----------|----------|----------|
| 1~3품 패 | 50% | 20% |
| 4~6품 패 | 30% | 30% |
| 7~9품 패 | 15% | 30% |
| 10~12품 패 | 5% | 15% |
| 13품+ 패 | - | 5% |

> 13품 이상 패는 개봉으로 직접 획득 불가.
> **조건부 해금**을 통해서만 획득 가능 → 연마의 가치를 보장.

### 천장(保底) 시스템
- 일반 개봉 30회: 7~9품 패 보장
- 고급 개봉 50회: 10~12품 패 보장
