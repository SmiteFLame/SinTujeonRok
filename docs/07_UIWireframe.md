### 패결 화면 요소

| 영역 | 설명 |
|------|------|
| **상단** | 관문 번호, 현재 수/최대 수, 배속 표시 |
| **적 영역** | 적 유닛 + 개별 기력 바, 상태이상 아이콘 |

| **합/비기** | 활성 중인 합(合)/비기 목록 |
| **배속** | 1x/2x/3x 선택, 일시정지 버튼 |

### 조작 방법
```
- 보유 패 터치 → 패진 빈 슬롯에 배치
- 패진 패 터치 → 해제 (보유 목록으로 돌아감)
- 패진 패 길게 누르기 → 패 상세 정보
- 패진 내 드래그 → 배치 순서 변경
- 필터: 전체 / 패 종류별 / 품계별
- 정렬: 전투력 / 연마 / 품계 / 최근 획득
```

## 13. UI 공통 규칙

### 레이아웃 기준
```
- 해상도: 1080×1920 기준 (9:16 세로)
- 안전 영역: 상단 80px, 하단 120px (노치/하단 바)
- 하단 네비게이션: 고정 120px
- 상단 바: 고정 100px
- 콘텐츠 영역: 나머지 (약 1700px)
```

### 터치 규칙
```
- 최소 터치 영역: 44×44dp (약 132px @3x)
- 버튼 간 최소 간격: 8dp
- 길게 누르기: 0.5초 이상 → 상세 정보 표시
- 스와이프: 좌우 → 탭 전환, 상하 → 스크롤
```

### 팝업/모달
```
- 배경 딤 처리 (50% 반투명 검정)
- 외부 터치 시 닫기
- 확인/취소 필요한 팝업은 외부 터치 무시
- 애니메이션: 아래→위 슬라이드 (200ms)
```

### 색상 시스템
```
- 배경: 한지 색 (#F5E6C8)
- 주요 액센트: 먹색 (#2C2C2C)
- 보조 액센트: 주홍 (#D4380D)
- 버튼 (긍정): 남색 (#1A3A5C)
- 버튼 (경고): 적색 (#B71C1C)
- 품계 색상: Terminology.md 참조
  백→청→남→자→금→적→천→현
```
