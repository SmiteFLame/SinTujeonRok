## 2. 각 루프 단계 상세

### ① 방치/수련 (오프라인 루프)

플레이어가 게임을 종료해도 자동으로 진행되는 핵심 시스템.

| 항목 | 설명 |
|------|------|
| **수련 대상** | 현재 선택된 관문(關門)의 잡몹을 자동 반복 사냥 |
| **획득 자원** | 은냥(銀兩), 수련치(修鍊値), 패편(牌片) |
| **시간 계산** | 접속 시 `(현재시간 - 마지막접속시간)` 으로 보상 계산 |
| **최대 방치** | 12시간 (이후 수련 정지, 접속 유도) |
| **효율** | 방치 수련은 액티브 플레이 대비 **50~70%** 효율 |

#### 방치 보상 공식
```
보상 = 분당_기본획득량 × 경과시간(분) × 관문_보정값 × 가호_배율

예시:
- 분당 기본: 은냥 10, 수련치 5
- 경과 시간: 480분 (8시간)
- 관문 보정: 1.5 (관문 15)
- 가호 배율: 1.0 (기본)
→ 은냥: 10 × 480 × 1.5 × 1.0 = 7,200 은냥
→ 수련치: 5 × 480 × 1.5 × 1.0 = 3,600 수련치
```

#### 패편(牌片) 드랍
```
패편 드랍 = floor(경과시간(분) / 드랍간격(분)) × 드랍개수

- 범패(凡牌) 패편: 10분마다 1개
- 수(秀) 패편: 30분마다 1개
- 걸(傑) 패편: 120분마다 1개 (확률 50%)
- 신물(神物) 패편: 드랍 없음 (개봉/두목 전용)
```

### ② 접속 → 보상 수령

| 항목 | 설명 |
|------|------|
| **보상 팝업** | 접속 시 방치 보상 요약 화면 표시 |
| **광고 부스터** | 광고 시청 시 방치 보상 **2배** |
| **빠른 수령** | 원터치로 모든 보상 수령 |
| **접속 보너스** | 매일 첫 접속 시 추가 보상 (문안(問安)) |

#### 문안(問安) 보상표
| 일차 | 보상 |
|------|------|
| 1일 | 은냥 500 |
| 2일 | 범패(凡牌) 1장 |
| 3일 | 은냥 1,000 |
| 4일 | 수(秀) 패편 5개 |
| 5일 | 은냥 2,000 |
| 6일 | 걸(傑) 패편 3개 |
| 7일 | **개봉권 1장** |

### ③ 패 수집 / 연마(鍊磨) / 각성(覺醒)

#### 패 획득 경로
| 경로 | 설명 |
|------|------|
| **방치 수련** | 범/수 품계 패편 자동 획득 |
| **관문 돌파** | 첫 돌파 보상으로 특정 패 획득 |
| **개봉(開封)** | 옥패(玉牌)로 걸/신 품계 획득 가능 |
| **패편 합성** | 패편(牌片) 일정 수 모으면 완성 패로 합성 |
| **두목(頭目) 드랍** | 두목 처치 시 한정 패 드랍 |
| **인연(因緣)/절기(節氣)** | 기간 한정 패 |

#### 연마(鍊磨) 시스템
```
연마 단계: +0 ~ +10
비용: 은냥 + 동일 패 종류 (재료)

+0 → +1: 은냥 100,  성공률 100%
+1 → +2: 은냥 200,  성공률 100%
+2 → +3: 은냥 400,  성공률 100%
+3 → +4: 은냥 800,  성공률 90%
+4 → +5: 은냥 1,600, 성공률 80%
+5 → +6: 은냥 3,200, 성공률 70%
+6 → +7: 은냥 6,400, 성공률 60%
+7 → +8: 은냥 12,800, 성공률 50%
+8 → +9: 은냥 25,600, 성공률 40%
+9 → +10: 은냥 51,200, 성공률 30%

실패 시: 연마 단계 유지 (하락 없음), 재료만 소모
```

#### 각성(覺醒) 시스템
```
범(凡) → 수(秀): 같은 패 3장 + 은냥 5,000
수(秀) → 걸(傑): 같은 패 3장 + 은냥 20,000
걸(傑) → 신(神): 같은 패 3장 + 은냥 100,000 + 각성석(覺醒石)

각성 시: 연마 단계 초기화, 기본 스탯 대폭 상승
```

### ④ 패진(牌陣) 편성 (전략 구성)

#### 기본 규칙
| 항목 | 값 |
|------|-----|
| 패진 크기 | **8장** |
| 같은 패 종류 제한 | 최대 3장 |
| 같은 패 중복 | 불가 (동일 패 2장 X) |
| 패세(牌勢) 슬롯 | 최대 5개 (프리셋 저장) |

#### 합(合) 보너스
같은 종류의 패를 여러 장 넣으면 합(合)이 발동합니다.

| 조건 | 합 효과 |
|------|---------|
| 같은 종류 2장 | **소합(小合)** - 해당 속성 능력 +15% |
| 같은 종류 3장 | **대합(大合)** - 해당 속성 능력 +35% + 특수 효과 |

#### 합(合) 특수 효과 예시
| 종류 | 대합 특수 효과 |
|------|---------------|
| 인(人) 3장 | **인해전술** - 공(攻) 시 30% 확률로 추가 타격 |
| 어(魚) 3장 | **물흐르듯** - 피격 시 25% 확률로 회피(回避) |
| 조(鳥) 3장 | **연환격** - 매 수(手) 2회 공격 |
| 치(雉) 3장 | **천둥벌거숭이** - 범위 공격이 전체 공격으로 확장 |
| 성(星) 3장 | **천문관측** - 모든 가호(加護) 지속시간 2배 |
| 마(馬) 3장 | **천리질주** - 첫 수(手) 데미지 2배 |
| 장(獐) 3장 | **맹독(猛毒)** - 저주(詛呪) 효과 50% 증가 |
| 토(兎) 3장 | **달토끼** - 매 수(手) 기력(氣力) 5% 자동 회복 |

#### 비기(秘技) - 특수 조합
서로 다른 종류의 패 조합으로 강력한 비기가 발동됩니다.

| 비기 이름 | 조합 | 효과 |
|----------|------|------|
| **천지인(天地人)** | 인 + 성 + 마 | 전체 스탯 20% 증가 |
| **산군포효(山君咆哮)** | 인 + 마 + 장 | 공(攻) 40% 증가 + 적 수(守) 감소 |
| **풍림화산(風林火山)** | 조 + 장 + 마 + 치 | 4종 원소 공격 (매 수(手) 랜덤) |
| **음양조화(陰陽調和)** | 성 + 토 + 어 | 피격 시 반격 + 기력(氣力) 회복 |
| **팔목개화(八目開花)** | 8종 각 1장씩 | **궁극 비기** - 모든 스탯 30% + 매 수(手) 랜덤 특수효과 |

### ⑤ 패결(牌決) (자동 전투)

#### 패결 흐름
```
패결 개시
    │
    ▼
[수(手) 시작] ←──────────────────┐
    │                            │
    ▼                            │
[패사의 패 순서대로 술(術) 발동]   │
    │                            │
    ▼                            │
[적 행동]                        │
    │                            │
    ▼                            │
[수(手) 종료 처리]                │
    │                            │
    ├── 적 전멸? → 승(勝)!        │
    ├── 패사 기력 소진? → 패(敗)   │
    └── 계속 → ─────────────────┘
```

#### 패결 규칙 요약
| 항목 | 설명 |
|------|------|
| 방식 | **수(手)제 자동 패결** |
| 패 발동 | 패진 순서대로 매 수(手) 1~2장 자동 발동 |
| 수 제한 | 최대 30수 (초과 시 패(敗)) |
| 속도 조절 | 1x / 2x / 3x 배속 |
| 스킵 | 돌파한 관문은 즉시 완료 가능 |

## 3. 성장 곡선

### 패사 품계(品階) 진행
```
1~3품 (한양):     빠른 성장 (튜토리얼 구간)     → 약 1일
4~6품 (경기):     보통 성장 (초반 콘텐츠)        → 약 3~5일
7~9품 (호서):     느린 성장 (중반 콘텐츠)        → 약 1~2주
10~15품 (호남/영남): 매우 느린 성장 (후반 콘텐츠) → 약 3~4주
16~24품 (관북~):  극한 성장 (엔드 콘텐츠)        → 지속적
```

### 관문(關門) 진행 속도
```
관문 1~10:   튜토리얼, 쉽게 돌파               → 1일차
관문 11~30:  점진적 난이도 상승                → 1주차
관문 31~60:  패진 전략이 중요해지는 구간        → 2~3주차
관문 61~100: 연마/각성 필수                    → 1~2개월차
관문 100+:   엔드 콘텐츠 (무한)                → 지속적
```

## 4. 재화 시스템

| 재화 | 획득 방법 | 용도 |
|------|----------|------|
| **은냥(銀兩)** | 방치, 패결, 일과 | 연마, 각성, 객주(客主) 구매 |
| **옥패(玉牌)** | 업적, 문안, 과금 | 개봉(開封), 특수 아이템 |
| **수련서(修鍊書)** | 방치, 인연 | 패사 수련치 부스트 |
| **패편(牌片)** | 방치, 패결 드랍 | 패 합성 재료 |
| **각성석(覺醒石)** | 두목, 인연 | 걸→신 각성 재료 |

### 무과금 유저 일일 획득량 (예상)
| 재화 | 일일 획득 | 비고 |
|------|----------|------|
| 은냥 | ~15,000 | 방치 12h + 패결 + 일과 |
| 옥패 | ~30 | 일과 + 광고 |
| 범 패편 | ~70 | 방치 기준 |
| 수 패편 | ~25 | 방치 기준 |
| 걸 패편 | ~3 | 방치(확률) + 두목 |

## 5. 일일 콘텐츠 사이클

### 일과(日課) (매일 초기화)
| 일과 | 보상 |
|------|------|
| 접속하기 | 은냥 500 |
| 방치 보상 수령 | 은냥 300 |
| 패 연마 3회 | 수련서 1개 |
| 관문 5회 돌파 | 은냥 1,000 |
| 패진 편성 1회 | 패편(랜덤) 3개 |
| 일일 시련 도전 | 옥패 20 |
| **모든 일과 완료** | **옥패 10 + 개봉권 조각** |

### 주간 콘텐츠
| 콘텐츠 | 주기 | 보상 |
|--------|------|------|
| 주간 두목 | 매주 월요일 갱신 | 걸 품계 패편 + 각성석 |
| 투전탑 절기 | 매주 일요일 정산 | 랭킹 보상 (옥패 + 한정 스킨) |

## 6. 리텐션 설계

### 재접속 유도 장치
| 장치 | 설명 |
|------|------|
| **방치 보상 한도** | 12시간 후 수련 정지 → 재접속 유도 |
| **문안(問安)** | 연속 문안 보상 증가 (7일 사이클) |
| **일일 시련** | 매일 다른 룰 → 매일 도전하고 싶은 동기 |
| **푸시 알림** | "방치 보상이 가득 찼습니다" (12시간 후) |
| **일일 무료 개봉** | 매일 1회 무료 개봉(開封) |
| **길드/친구** | 사회적 유대감 (추후 업데이트) |

### 이탈 방지
| 상황 | 대응 |
|------|------|
| 패결에서 연속 패(敗) | 난이도 힌트 + 추천 패진 제안 |
| 3일 미접속 | 복귀 보상 패키지 (은냥 + 옥패) |
| 7일 미접속 | 대형 복귀 보상 + 신규 패 지급 |
| 콘텐츠 소진 | 절기/인연 업데이트로 지속 콘텐츠 공급 |
