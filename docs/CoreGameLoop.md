## 2. 각 루프 단계 상세

### ① 방치/수련 (오프라인 루프)

플레이어가 게임을 종료해도 자동으로 진행되는 핵심 시스템.

| 항목 | 설명 |
|------|------|
| **수련 대상** | 현재 선택된 스테이지의 잡몹을 자동 반복 사냥 |
| **획득 자원** | 은냥(기본재화), 수련경험치, 투전패 조각 |
| **시간 계산** | 접속 시 `(현재시간 - 마지막접속시간)` 으로 보상 계산 |
| **최대 방치** | 12시간 (이후 수련 정지, 접속 유도) |
| **효율** | 방치 수련은 액티브 플레이 대비 **50~70%** 효율 |

#### 방치 보상 공식
```
보상 = 분당_기본획득량 × 경과시간(분) × 스테이지_보정값 × 버프_배율

예시:
- 분당 기본: 은냥 10, 경험치 5
- 경과 시간: 480분 (8시간)
- 스테이지 보정: 1.5 (스테이지 15)
- 버프 배율: 1.0 (기본)
→ 은냥: 10 × 480 × 1.5 × 1.0 = 7,200 은냥
→ 경험치: 5 × 480 × 1.5 × 1.0 = 3,600 경험치
```

#### 투전패 조각 드랍
```
조각 드랍 = floor(경과시간(분) / 드랍간격(분)) × 드랍개수

- 범(凡) 조각: 10분마다 1개
- 수(秀) 조각: 30분마다 1개
- 걸(傑) 조각: 120분마다 1개 (확률 50%)
- 신(神) 조각: 드랍 없음 (뽑기/보스 전용)
```

### ② 접속 → 보상 수령

| 항목 | 설명 |
|------|------|
| **보상 팝업** | 접속 시 방치 보상 요약 화면 표시 |
| **광고 부스터** | 광고 시청 시 방치 보상 **2배** |
| **빠른 수령** | 원터치로 모든 보상 수령 |
| **접속 보너스** | 매일 첫 접속 시 추가 보상 (출석 체크) |

#### 출석 체크 보상표
| 일차 | 보상 |
|------|------|
| 1일 | 은냥 500 |
| 2일 | 범(凡) 투전패 1장 |
| 3일 | 은냥 1,000 |
| 4일 | 수(秀) 투전패 조각 5개 |
| 5일 | 은냥 2,000 |
| 6일 | 걸(傑) 투전패 조각 3개 |
| 7일 | **뽑기권 1장** |

### ③ 카드 수집 / 강화 / 진화

#### 카드 획득 경로
| 경로 | 설명 |
|------|------|
| **방치 수련** | 범/수 등급 조각 자동 획득 |
| **스테이지 클리어** | 첫 클리어 보상으로 특정 카드 획득 |
| **뽑기** | 프리미엄 재화(옥패)로 걸/신 등급 획득 가능 |
| **조각 합성** | 조각 일정 수 모으면 완성 카드로 합성 |
| **보스 드랍** | 보스 처치 시 한정 카드 드랍 |
| **이벤트/시즌** | 기간 한정 카드 |

#### 카드 강화 시스템
```
강화 단계: +0 ~ +10
비용: 은냥 + 동일 패 종류 카드 (재료)

+0 → +1: 은냥 100,  성공률 100%
+1 → +2: 은냥 200,  성공률 100%
+2 → +3: 은냥 400,  성공률 100%
+3 → +4: 은냥 800,  성공률 90%
+4 → +5: 은냥 1,600, 성공률 80%
+5 → +6: 은냥 3,200, 성공률 70%
+6 → +7: 은냥 6,400, 성공률 60%
+7 → +8: 은냥 12,800, 성공률 50%
+8 → +9: 은냥 25,600, 성공률 40%
+9 → +10: 은냥 51,200, 성공률 30%

실패 시: 강화 단계 유지 (하락 없음), 재료만 소모
```

#### 카드 진화 시스템
```
범(凡) → 수(秀): 같은 카드 3장 + 은냥 5,000
수(秀) → 걸(傑): 같은 카드 3장 + 은냥 20,000
걸(傑) → 신(神): 같은 카드 3장 + 은냥 100,000 + 특수재료

진화 시: 강화 단계 초기화, 기본 스탯 대폭 상승
```

### ④ 덱 편집 (전략 구성)

#### 기본 규칙
| 항목 | 값 |
|------|-----|
| 덱 크기 | **8장** |
| 같은 패 종류 제한 | 최대 3장 |
| 같은 카드 중복 | 불가 (동일 카드 2장 X) |
| 덱 슬롯 | 최대 5개 (프리셋 저장) |

#### 시너지 보너스
같은 종류의 패를 여러 장 넣으면 시너지가 발동합니다.

| 조건 | 시너지 효과 |
|------|------------|
| 같은 종류 2장 | **소(小) 시너지** - 해당 속성 능력 +15% |
| 같은 종류 3장 | **대(大) 시너지** - 해당 속성 능력 +35% + 특수 효과 |

#### 시너지 특수 효과 예시
| 종류 | 대시너지 특수 효과 |
|------|------------------|
| 인(人) 3장 | **인해전술** - 공격 시 30% 확률로 추가 타격 |
| 어(魚) 3장 | **물흐르듯** - 피격 시 25% 확률로 완전 회피 |
| 조(鳥) 3장 | **연환격** - 매 턴 2회 공격 |
| 치(雉) 3장 | **천둥벌거숭이** - 범위 공격이 전체 공격으로 확장 |
| 성(星) 3장 | **천문관측** - 모든 버프 지속시간 2배 |
| 마(馬) 3장 | **천리질주** - 첫 턴 데미지 2배 |
| 장(獐) 3장 | **맹독** - 디버프 효과 50% 증가 |
| 토(兎) 3장 | **달토끼** - 매 턴 체력 5% 자동 회복 |

#### 비기(秘技) - 특수 조합
서로 다른 종류의 패 조합으로 강력한 비기가 발동됩니다.

| 비기 이름 | 조합 | 효과 |
|----------|------|------|
| **천지인(天地人)** | 인 + 성 + 마 | 전체 스탯 20% 증가 |
| **산군포효(山君咆哮)** | 인 + 마 + 장 | 공격력 40% 증가 + 적 방어력 감소 |
| **풍림화산(風林火山)** | 조 + 장 + 마 + 치 | 4종 원소 공격 (매 턴 랜덤) |
| **음양조화(陰陽調和)** | 성 + 토 + 어 | 공격받을 때마다 반격 + 회복 |
| **팔목개화(八目開花)** | 8종 각 1장씩 | **궁극 비기** - 모든 스탯 30% + 매 턴 랜덤 특수효과 |

### ⑤ 패 대결 (자동 전투)

#### 전투 흐름
```
전투 시작
    │
    ▼
[턴 시작] ←──────────────────┐
    │                        │
    ▼                        │
[패사 카드 순서대로 발동]      │
    │                        │
    ▼                        │
[적 행동]                     │
    │                        │
    ▼                        │
[턴 종료 처리]                │
    │                        │
    ├── 적 전멸? → 승리!       │
    ├── 패사 사망? → 패배      │
    └── 계속 → ──────────────┘
```

#### 전투 규칙 요약
| 항목 | 설명 |
|------|------|
| 방식 | **턴제 자동 전투** |
| 카드 발동 | 덱 순서대로 매 턴 1~2장 자동 발동 |
| 턴 제한 | 최대 30턴 (초과 시 패배) |
| 속도 조절 | 1x / 2x / 3x 배속 |
| 스킵 | 클리어한 스테이지는 즉시 완료 가능 |

## 3. 성장 곡선

### 패사 레벨
```
레벨 1~10:   빠른 성장 (튜토리얼 구간)     → 약 1일
레벨 11~30:  보통 성장 (초반 콘텐츠)        → 약 3~5일
레벨 31~50:  느린 성장 (중반 콘텐츠)        → 약 1~2주
레벨 51~80:  매우 느린 성장 (후반 콘텐츠)    → 약 3~4주
레벨 81~100: 극한 성장 (엔드 콘텐츠)        → 지속적
```

### 스테이지 진행 속도
```
스테이지 1~10:   튜토리얼, 쉽게 클리어       → 1일차
스테이지 11~30:  점진적 난이도 상승          → 1주차
스테이지 31~60:  덱 전략이 중요해지는 구간    → 2~3주차
스테이지 61~100: 강화/진화 필수              → 1~2개월차
스테이지 100+:   엔드 콘텐츠 (무한)          → 지속적
```

## 4. 재화 시스템

| 재화 | 획득 방법 | 용도 |
|------|----------|------|
| **은냥(銀)** | 방치, 전투, 일일미션 | 강화, 진화, 상점 구매 |
| **옥패(玉牌)** | 업적, 출석, 과금 | 프리미엄 뽑기, 특수 아이템 |
| **수련서** | 방치, 이벤트 | 패사 경험치 부스트 |
| **투전패 조각** | 방치, 전투 드랍 | 카드 합성 재료 |
| **진화석** | 보스, 이벤트 | 걸→신 진화 재료 |

### 무과금 유저 일일 획득량 (예상)
| 재화 | 일일 획득 | 비고 |
|------|----------|------|
| 은냥 | ~15,000 | 방치 12h + 전투 + 미션 |
| 옥패 | ~30 | 일일미션 + 광고 |
| 범 조각 | ~70 | 방치 기준 |
| 수 조각 | ~25 | 방치 기준 |
| 걸 조각 | ~3 | 방치(확률) + 보스 |

## 5. 일일 콘텐츠 사이클

### 일일 미션 (매일 초기화)
| 미션 | 보상 |
|------|------|
| 접속하기 | 은냥 500 |
| 방치 보상 수령 | 은냥 300 |
| 카드 강화 3회 | 수련서 1개 |
| 스테이지 5회 클리어 | 은냥 1,000 |
| 덱 편집 1회 | 투전패 조각(랜덤) 3개 |
| 일일 시련 도전 | 옥패 20 |
| **모든 미션 완료** | **옥패 10 + 뽑기권 조각** |

### 주간 콘텐츠
| 콘텐츠 | 주기 | 보상 |
|--------|------|------|
| 주간 보스 | 매주 월요일 갱신 | 걸 등급 카드 조각 + 진화석 |
| 투전탑 시즌 | 매주 일요일 정산 | 랭킹 보상 (옥패 + 한정 스킨) |

## 6. 리텐션 설계

### 재접속 유도 장치
| 장치 | 설명 |
|------|------|
| **방치 보상 한도** | 12시간 후 수련 정지 → 재접속 유도 |
| **출석 체크** | 연속 출석 보상 증가 (7일 사이클) |
| **일일 시련** | 매일 다른 룰 → 매일 도전하고 싶은 동기 |
| **푸시 알림** | "방치 보상이 가득 찼습니다" (12시간 후) |
| **일일 무료 뽑기** | 매일 1회 무료 뽑기 |
| **길드/친구** | 사회적 유대감 (추후 업데이트) |

### 이탈 방지
| 상황 | 대응 |
|------|------|
| 전투에서 연속 패배 | 난이도 힌트 + 추천 덱 제안 |
| 3일 미접속 | 복귀 보상 패키지 (은냥 + 옥패) |
| 7일 미접속 | 대형 복귀 보상 + 신규 카드 지급 |
| 콘텐츠 소진 | 시즌/이벤트 업데이트로 지속 콘텐츠 공급 |
