## 1. 전체 아트 방향

### 핵심 키워드
```
"조선 수묵화 × 모던 캐주얼"

전통 수묵화/민화의 붓 터치와 먹의 느낌을 기반으로 하되,
모바일 캐주얼 게임에 맞는 깔끔하고 가독성 높은 스타일.
```

### 스타일 정의

| 항목 | 방향 |
|------|------|
| **전체 톤** | 한지(韓紙) 위에 수묵/담채 느낌 |
| **캐릭터** | 수묵화풍 세미 SD (3~4등신) |
| **배경** | 한국 전통 산수화/민화 스타일 |
| **패(牌)** | 투전 카드 형태, 전통 문양 + 캐릭터 일러스트 |
| **UI** | 한지/나무/먹물 텍스처 기반, 깔끔한 모던 레이아웃 |
| **이펙트** | 먹물 번짐, 붓 터치, 한지 찢어짐 |

### 레퍼런스 방향
```
참고할 느낌:
  - 한국 전통 민화의 해학적 느낌
  - 수묵화의 여백과 붓 터치
  - 일본 "오카미" 스타일의 먹 이펙트 (동양화 + 게임)
  - Balatro의 깔끔한 카드 UI
  - 방치형 게임의 직관적 UI

피해야 할 느낌:
  ✗ 서양 판타지 스타일
  ✗ 지나치게 사실적인 표현
  ✗ 과도한 애니메이션/3D 느낌
  ✗ 일본풍 이세카이 스타일
```

## 4. 캐릭터 디자인

### 패사(주인공) - 한수

```
- 체형: 세미 SD (3.5등신)
- 복장: 중인 차림 (도포 + 갓), 후반부 패사복으로 변경
- 특징: 투전패를 허리춤에 차고 다님
- 표정: 다양한 감정 표현 (기본/승리/패배/놀람/결의)
- 애니메이션: Idle(호흡), 승리 포즈, 패배 리액션
```

### 적 디자인 방향

| 유형 | 디자인 | 컨셉 |
|------|--------|------|
| **잡몹** | 복면 도적, 하급 흑패단원 | 간단, 실루엣으로 구분 가능 |
| **정예** | 두건 쓴 중급 흑패단원 | 잡몹 + 장식 추가 |
| **중간 두목** | 개성 있는 간부급 | 고유 무기/복장 |
| **장 두목** | 풀 일러스트 보스 | 컷인 연출용 대형 일러스트 |

### NPC 디자인

| NPC | 디자인 키워드 |
|-----|-------------|
| 장 노인 | 수염 긴 노인, 낡은 도포, 지혜로운 눈 |
| 월향 | 기생 차림, 부채, 날카로운 눈 |
| 박거한 | 양반 차림, 정갈, 오만한 표정 → 후반 부드러워짐 |
| 흑수 | 검은 도포, 얼굴 반 가림, 위압감 |

## 6. 이펙트 & 애니메이션

### 전투 이펙트

```
[공격]
  - 물리 공격: 먹물 튀김 + 붓 자국 슬래시
  - 범위 공격: 먹물 원형 번짐
  - 전체 공격: 화면 전체 먹물 파동

[속성별]
  인(人): 먹색 칼날 / 인형 실루엣
  어(魚): 파란 물결 / 파도
  조(鳥): 깃털 날림 / 바람 흔적
  치(雉): 번개 붓 터치 / 우레
  성(星): 금빛 별가루 / 별자리
  마(馬): 먹 질주 궤적 / 발자국
  장(獐): 보라 독기 / 안개
  토(兎): 녹색 치유 파티클 / 달빛

[비술]
  - 수묵화 컷인 (전체 화면 전환)
  - 한자 대형 표시 + 붓으로 쓰는 애니메이션
  - 먹물 폭발 + 화면 진동
```

### UI 애니메이션

```
[화면 전환]
  - 먹물 닦아내기 (와이프 트랜지션)
  - 한지 넘기기 (페이드)
  - 두루마리 펼침 (팝업 등장)

[보상 획득]
  - 재화: 위로 튀어오르며 카운트 증가
  - 패 획득: 봉인 해제 연출 (종이 찢어짐)
  - 신물 획득: 금빛 광선 + 특수 BGM

[패결]
  - 패 발동: 카드 앞으로 슬라이드 + 빛남
  - 데미지 숫자: 붓글씨 스타일 팝업
  - 급소: 빨간 대형 숫자 + 화면 흔들림
  - 승리: 도장 쾅 찍기 (勝 한자)
  - 패배: 먹물 번짐 + 어두워짐
```

## 7. 무료 에셋 활용 계획

### 개발 단계별 에셋 전략

```
Phase 1 (프로토타입):
  - 플레이스홀더 (단색 도형 + 텍스트)
  - 무료 폰트 적용
  - 기본 UI 레이아웃만 구현

Phase 2 (콘텐츠):
  - 무료 에셋 교체 + 최소 자체 제작
  - AI 이미지 생성 활용 (컨셉 아트 → 게임 에셋)
  - 무료 사운드 에셋

Phase 3 (폴리싱):
  - 핵심 에셋 자체 제작 or 외주
  - 나머지 무료/AI 에셋 품질 통일
```

### 무료 리소스 목록

| 카테고리 | 리소스 | 용도 |
|---------|--------|------|
| **폰트** | 나눔명조, Pretendard | 한글 UI |
| **폰트** | 네이버 마루부리 | 제목/한자 |
| **텍스처** | 한지/나무결 (CC0) | UI 배경 |
| **사운드** | freesound.org | SFX |
| **BGM** | 국악 CC0 / 자체 제작 | 배경음 |
| **아이콘** | Material Icons | 시스템 아이콘 |
| **이펙트** | Unity Particle System | 전투 이펙트 |

### AI 이미지 생성 활용

```
활용 범위:
  ✅ 패 일러스트 초안 → 수정/가공
  ✅ 배경 컨셉 아트
  ✅ NPC/적 디자인 초안
  ✅ UI 텍스처 (한지, 먹물 등)

주의:
  ✗ AI 생성물 그대로 사용 지양
  ✗ 스타일 일관성 수동 보정 필요
  ✗ 상용 이미지 생성 서비스 라이선스 확인
```

## 9. 사운드 방향 (간략)

### BGM
```
- 메인 로비: 가야금/거문고 선율, 편안한 국악풍
- 패결: 장구/북 리듬, 긴장감 있는 국악 퓨전
- 두목전: 대금 솔로 + 강한 타악, 긴박한 분위기
- 스토리: 피리/소금 선율, 감성적
```

### SFX
```
- 패 발동: 종이 펼치는 소리 + 바람
- 술 발동: 붓 휘두르는 소리
- 급소: 쨍 (금속음)
- 연마 성공: 딩 (맑은 종소리)
- 연마 실패: 둥 (탁한 북소리)
- 개봉: 봉인 해제 (종이 찢는 소리 + 빛)
- 승리: 관악기 팡파레
- 패배: 낮은 대금 소리
```
