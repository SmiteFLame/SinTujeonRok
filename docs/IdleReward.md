## 1. 방치 시스템 개요

### 핵심 철학
```
"접속하지 않아도 성장하고, 접속하면 더 빠르게 성장한다"

방치 보상 = 액티브 플레이의 50~70% 효율
→ 방치만으로도 충분하지만, 접속할 이유가 있다
```

### 방치 수련 흐름
```
게임 종료 (오프라인)
    │
    ▼
자동 수련 시작 (현재 관문 잡몹 반복)
    │
    ├─ 은냥(銀兩) 누적
    ├─ 수련치(修鍊値) 누적
    └─ 패편(牌片) 누적
    │
    ▼
재접속
    │
    ▼
방치 보상 팝업 → 수령 (광고 시청 시 2배)
```

## 2. 시간별 보상 계산

### 기본 공식

```
보상 = 분당_기본량 × 경과시간(분) × 관문_보정 × 가호_배율 × 공력_보정

경과시간 = min(현재시간 - 마지막접속시간, 720분)
  └─ 최대 12시간(720분) 제한
```

### 분당 기본 획득량 (기본값)

| 재화 | 분당 기본량 | 1시간 | 4시간 | 8시간 | 12시간(최대) |
|------|-----------|-------|-------|-------|-------------|
| **은냥(銀兩)** | 10 | 600 | 2,400 | 4,800 | 7,200 |
| **수련치(修鍊値)** | 5 | 300 | 1,200 | 2,400 | 3,600 |
| **범 패편** | 0.1 (10분당 1) | 6 | 24 | 48 | 72 |
| **수 패편** | 0.033 (30분당 1) | 2 | 8 | 16 | 24 |
| **걸 패편** | 0.004 (120분당 50%) | 0~1 | 1 | 2 | 3 |

> 신물(神物) 패편은 방치 드랍 불가 (개봉/두목 전용)

### 관문(關門) 보정

현재 수련 중인 관문에 따라 보상이 증가합니다.

```
관문_보정 = 1.0 + (관문번호 - 1) × 0.05

관문  1: ×1.00     관문 21: ×2.00
관문  5: ×1.20     관문 30: ×2.45
관문 10: ×1.45     관문 50: ×3.45
관문 15: ×1.70     관문 100: ×5.95
```

#### 관문별 12시간 방치 보상 예시

| 관문 | 보정 | 은냥 | 수련치 | 범 패편 | 수 패편 |
|------|------|------|--------|---------|---------|
| 1 | ×1.00 | 7,200 | 3,600 | 72 | 24 |
| 10 | ×1.45 | 10,440 | 5,220 | 104 | 34 |
| 30 | ×2.45 | 17,640 | 8,820 | 176 | 58 |
| 50 | ×3.45 | 24,840 | 12,420 | 248 | 82 |
| 100 | ×5.95 | 42,840 | 21,420 | 428 | 142 |

### 공력(功力) 보정

패사의 총 공력에 따라 방치 효율이 소폭 증가합니다.

```
공력_보정 = 1.0 + 총공력 × 0.001

총 공력 50:   ×1.05  (+5%)
총 공력 100:  ×1.10  (+10%)
총 공력 300:  ×1.30  (+30%)
총 공력 500:  ×1.50  (+50%)
```

## 3. 방치 vs 액티브 보상 비교

### 효율 비교 (1시간 기준, 관문 30)

| 활동 | 은냥 | 수련치 | 패편 | 추가 보상 |
|------|------|--------|------|----------|
| **방치 수련** | 1,470 | 735 | 범14, 수5 | - |
| **액티브 (관문 반복)** | 2,500 | 1,200 | 범20, 수8, 걸1 | 두목 드랍 확률 |
| **액티브 효율** | **×1.70** | **×1.63** | **×1.4~2.0** | 걸 패편 추가 |

### 설계 의도
```
방치: 기본 성장 보장 (시간 없을 때도 진행)
액티브: 더 빠른 성장 + 추가 보상 (두목 드랍, 걸 패편, 인연 진행)

→ 바쁜 유저: 방치로 천천히 진행 (불이익 없음)
→ 열정 유저: 액티브로 빠르게 진행 (보상감)
→ 과금 유저: 방치 부스터로 방치 효율 = 액티브 수준
```

## 4. 접속 유도 시스템

### 4-1. 문안(問安) - 일일 출석

```
7일 주기 순환. 연속 문안 시 보상 증가.
문안 끊김 시: 1일차부터 다시 시작
```

| 일차 | 보상 | 누적 가치 |
|------|------|----------|
| 1일 | 은냥 500 | 500 |
| 2일 | 범패 1장 | ~1,000 |
| 3일 | 은냥 1,000 | 2,500 |
| 4일 | 수 패편 5개 | ~4,000 |
| 5일 | 은냥 2,000 | 6,500 |
| 6일 | 걸 패편 3개 | ~9,500 |
| 7일 | **개봉권 1장** | **~14,500** |

### 4-2. 일과(日課) - 일일 미션

| 일과 | 조건 | 보상 | 난이도 |
|------|------|------|--------|
| 문안 | 접속 | 은냥 500 | ★ |
| 보상 수령 | 방치 보상 수령 | 은냥 300 | ★ |
| 연마 3회 | 패 연마 3회 수행 | 수련서 1개 | ★★ |
| 관문 돌파 5회 | 관문 5회 승리 | 은냥 1,000 | ★★ |
| 패진 편성 | 패진 1회 변경 | 패편(랜덤) 3개 | ★ |
| 시련 도전 | 일일 시련 1회 | 옥패 20 | ★★★ |
| **전 일과 완료** | 위 6개 모두 | **옥패 10 + 개봉권 조각 3개** | - |

```
개봉권 조각 10개 = 개봉권 1장
→ 매일 일과 완료 시 약 3.3일마다 무료 개봉 1회
```

### 4-3. 주간 콘텐츠

| 콘텐츠 | 주기 | 보상 |
|--------|------|------|
| 주간 두목 | 월요일 갱신 | 걸 패편 5 + 각성석 1 |
| 투전탑 절기 | 일요일 정산 | 랭킹 보상 (옥패 + 한정 스킨) |
| 패사 대전 정산 | 일요일 정산 | 등급별 옥패/은냥 |
| 주간 일과 보상 | 일과 5일 이상 달성 | 옥패 30 + 수련서 5 |

### 4-4. 복귀 보상

| 미접속 기간 | 복귀 보상 |
|------------|----------|
| 1~2일 | 없음 (방치 보상만) |
| 3일 | 은냥 5,000 + 옥패 10 |
| 7일 | 은냥 15,000 + 옥패 30 + 범패 5장 |
| 14일 | 은냥 30,000 + 옥패 50 + 수 패 3장 + 개봉권 1장 |
| 30일+ | 은냥 50,000 + 옥패 100 + 걸 패 1장 + 개봉권 3장 |

## 5. 방치 보상 한도 & 접속 유도

### 최대 방치 시간

```
기본 최대: 12시간 (720분)
→ 12시간 이후 수련 자동 정지
→ 푸시 알림: "방치 보상이 가득 찼습니다!"

VIP/월정액 보너스:
→ 최대 방치 24시간으로 확장 (무과금 대비 ×2)
```

### 시간대별 접속 유도

```
아침 (06~09): 방치 보상 수령 (밤새 수련 보상)
점심 (12~14): 일과 진행, 빠른 관문 돌파
저녁 (18~21): 메인 플레이 (두목전, 시련, 패사 대전)
밤 (22~24):   방치 수련 시작 + 다음 날 준비

이상적 접속 패턴: 하루 2~3회, 총 15~30분
```

## 6. 광고 보상

### 광고 유형

| 유형 | 조건 | 보상 | 일일 제한 |
|------|------|------|----------|
| **방치 보상 2배** | 보상 수령 시 선택 | 방치 보상 ×2 | 2회 |
| **무료 개봉** | 개봉소에서 선택 | 개봉 1회 (범~수 품계) | 3회 |
| **은냥 보너스** | 객주(客主)에서 선택 | 은냥 1,000 즉시 지급 | 5회 |
| **수련 부스터** | 방치 시작 시 선택 | 다음 방치 보상 +50% | 1회 |
| **즉시 완료** | 관문 화면에서 선택 | 관문 3회 즉시 완료 | 3회 |

### 광고 보상 밸런스
```
무과금 유저 일일 광고 수익 (모든 광고 시청 시):
  방치 2배 ×2:  ~은냥 7,200 추가 (12시간 기준)
  무료 개봉 ×3:  개봉 3회 (범~수 패)
  은냥 보너스 ×5: 은냥 5,000
  수련 부스터 ×1: ~은냥 3,600 추가
  즉시 완료 ×3:   관문 9회 즉시 완료

총 일일 광고 시청: ~14회 (각 15~30초)
총 광고 시청 시간: ~5~7분
```

### 주요 콘텐츠 소요 재화 vs 획득량

| 소비 행위 | 소요 재화 | 획득까지 걸리는 시간 |
|----------|----------|-------------------|
| 연마 +5 (1장) | 은냥 ~2,500 | ~1시간 방치 |
| 연마 +10 (1장) | 은냥 ~10,000 | ~5시간 방치 |
| 각성 범→수 | 은냥 5,000 + 같은 패 3장 | 1~2일 |
| 각성 수→걸 | 은냥 20,000 + 같은 패 3장 | 3~5일 |
| 각성 걸→신 | 은냥 100,000 + 각성석 | 1~2주 |
| 고급 개봉 1회 | 옥패 300 | ~10일 (무과금) |
| 고급 개봉 천장 (50회) | 옥패 15,000 | ~500일 or 과금 |

### 밸런스 체크포인트

```
Day 1:   관문 1~10 돌파, 범패 8장 수집, 첫 연마 체험
Day 3:   관문 15~20, 수 패편으로 첫 각성, 패진 전략 시작
Day 7:   관문 25~30, 수 패 다수 보유, 합(合) 전략 본격화
Day 14:  관문 40~50, 걸 패 1~2장, 기운 해금 시작 (6품)
Day 30:  관문 60~70, 걸 패 다수, 두목전 본격 도전
Day 60:  관문 80~100, 스토리 후반, 신물 해금 시도
Day 90:  엔드 콘텐츠 (투전탑, 패사 대전, 시련)
```
