## 1. 패결 흐름

### 전체 플로우

```
┌─────────────────────────────────────────────────────────┐
│                    [패결 개시]                            │
│                        │                                │
│   ┌─ 패진(牌陣) 로드 (아군 8장, 적 N체)                  │
│   ├─ 합(合)/비기(秘技) 판정 → 패결 시작 시 즉시 적용     │
│   └─ 기운(氣韻) "패결 시작 시" 발동                      │
│                        │                                │
│                        ▼                                │
│   ┌─────────── [수(手) 시작] ◄──────────────┐           │
│   │                    │                    │           │
│   │   1. 수 시작 처리                        │           │
│   │      - 상태이상 틱 (화독/맹독 데미지)     │           │
│   │      - 가호/저주 지속수 감소             │           │
│   │      - 경직/속박 해제 판정               │           │
│   │                    │                    │           │
│   │   2. 행동 순서 결정                      │           │
│   │      - 전체 유닛 신(速) 기준 정렬        │           │
│   │      - 경직/속박 시 행동 스킵            │           │
│   │                    │                    │           │
│   │   3. 각 유닛 행동 (신 순서대로)           │           │
│   │      ├─ 비술 게이지 만충? → 비술 발동    │           │
│   │      ├─ 술(術) 대기 완료? → 술 발동      │           │
│   │      └─ 둘 다 아님 → 기본 공격           │           │
│   │                    │                    │           │
│   │   4. 데미지/효과 처리                    │           │
│   │      - 회피 판정 → 데미지 계산           │           │
│   │      - 급소 판정 → 추가 데미지           │           │
│   │      - 기운(氣韻) 조건부 발동            │           │
│   │                    │                    │           │
│   │   5. 수 종료 판정                        │           │
│   │      ├─ 적 전멸? → [승(勝)]              │           │
│   │      ├─ 아군 전멸? → [패(敗)]            │           │
│   │      ├─ 30수 초과? → [패(敗)]            │           │
│   │      └─ 계속 → ─────────────────────────┘           │
│   │                                                     │
│   └─────────────────────────────────────────────────────┘
│                                                         │
│                   [패결 종료]                             │
│   ├─ 승(勝): 보상 획득 (은냥, 수련치, 패편, 드랍)        │
│   └─ 패(敗): 보상 없음, 패진 재편성 유도                 │
└─────────────────────────────────────────────────────────┘
```

### 수(手) 내부 처리 순서 (요약)

```
[수 시작]
  → 상태이상 틱 처리
  → 가호/저주 지속수 -1
  → 행동 순서 결정 (신(速) 내림차순)
  → 유닛별 행동 (술/비술/기본공격)
  → 데미지/효과 적용
  → 기력 0 이하 유닛 퇴장
[수 종료]
```

## 2. 행동 순서 시스템

### 신(速) 기반 정렬

매 수(手)마다 **모든 유닛(아군 패 + 적)**을 신(速) 내림차순으로 정렬합니다.

```
행동 순서 = 신(速) × (1 + 가호/저주 보정) × 상태이상 보정

예시:
  장사(신 85) > 적A(신 80) > 참새(신 75) > 적B(신 60) > 붕어(신 50)
  → 장사 → 적A → 참새 → 적B → 붕어 순서로 행동
```

### 동일 신(速) 처리
```
신(速) 동일 시 우선순위:
1. 아군 우선
2. 패진 내 앞쪽 배치 우선
3. 랜덤
```

### 패 발동 방식
```
매 수(手)마다:
  - 각 패는 자신의 차례에 1회 행동
  - 술(術) 재사용 대기 완료 시 → 술 발동
  - 대기 중이면 → 기본 공격 (공(攻) × 1.0 데미지)
  - 비술(秘術) 게이지 만충 시 → 술 대신 비술 우선 발동
```

## 3. 데미지 계산 공식

### 기본 데미지

```
데미지 = (공격측 공(攻) × 술_위력배율) × (1 - 방어율) × 랜덤보정

방어율 = 방어측 수(守) / (방어측 수(守) + 상수K)
  └─ 상수K = 500 (방어율이 무한히 100%에 도달하지 못하도록)

랜덤보정 = 0.9 ~ 1.1 (±10% 랜덤)

최소 데미지 = 1 (0 이하 방지)
```

#### 방어율 예시
```
수(守) 100  → 방어율 16.7% → 데미지 83.3%
수(守) 250  → 방어율 33.3% → 데미지 66.7%
수(守) 500  → 방어율 50.0% → 데미지 50.0%
수(守) 1000 → 방어율 66.7% → 데미지 33.3%
수(守) 2000 → 방어율 80.0% → 데미지 20.0%
```

### 급소(急所)

```
급소 확률: 기본 5% (기운/가호로 증가 가능, 최대 50%)
급소 배율: 1.5배 (기운/가호로 증가 가능, 최대 2.5배)

급소 데미지 = 기본 데미지 × 급소_배율
```

### 속성 상성 (팔목 유불리)

팔목 간 **순환 상성** 관계:

```
┌─────────────────────────────────────┐
│                                     │
│   인(人) ──► 어(魚) ──► 조(鳥) ──┐  │
│     ▲                            │  │
│     │   마(馬) ──► 장(獐) ──┐    │  │
│     │     ▲                 │    │  │
│     │     │                 ▼    ▼  │
│   토(兎) ◄── 성(星) ◄── 치(雉)     │
│                                     │
└─────────────────────────────────────┘

유리 상성: 데미지 ×1.25 (+25%)
불리 상성: 데미지 ×0.80 (-20%)
동일/무관: 데미지 ×1.00
```

#### 상성표
| 공격 ↓ \ 수비 → | 인 | 어 | 조 | 치 | 성 | 마 | 장 | 토 |
|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
| **인(人)** | - | ▲ | - | - | - | - | - | ▼ |
| **어(魚)** | ▼ | - | ▲ | - | - | - | - | - |
| **조(鳥)** | - | ▼ | - | ▲ | - | - | - | - |
| **치(雉)** | - | - | ▼ | - | ▲ | - | - | - |
| **성(星)** | - | - | - | ▼ | - | - | - | ▲ |
| **마(馬)** | - | - | - | - | - | - | ▲ | - |
| **장(獐)** | - | - | - | - | - | ▼ | - | - |
| **토(兎)** | ▲ | - | - | - | ▼ | - | - | - |

> ▲ = 유리(×1.25) | ▼ = 불리(×0.80) | - = 보통(×1.00)

### 최종 데미지 공식 (종합)

```
최종 데미지 = 기본 데미지 × 속성_보정 × 급소_보정 × 가호/저주_보정

기본 데미지 = (공(攻) × 술_위력배율) × (1 - 방어율) × 랜덤보정(0.9~1.1)
속성 보정  = 상성에 따라 0.80 / 1.00 / 1.25
급소 보정  = 급소 발동 시 1.5 (기본), 아닐 시 1.0
가호/저주  = 공 증가 가호, 수 감소 저주 등 합산
```

## 4. 술(術) 시스템

### 술 유형

| 유형 | 설명 | 예시 |
|------|------|------|
| **단일 공(攻)** | 적 1체에 강한 데미지 | 장사 - 강타 |
| **범위 공(攻)** | 적 2~3체에 데미지 | 꿩 - 날갯짓 |
| **전체 공(攻)** | 적 전체에 데미지 | 독수리 - 폭풍 |
| **회복** | 아군 기력(氣力) 회복 | 거북이 - 장수 |
| **가호(加護)** | 아군에게 가호 부여 | 견우성 - 근면 |
| **저주(詛呪)** | 적에게 저주/상태이상 부여 | 도깨비 - 장난 |
| **복합** | 공격 + 부가효과 | 포졸 - 포박 (공격+속박) |

### 술 위력 등급

| 등급 | 위력 배율 | 재사용 대기 | 설명 |
|------|----------|------------|------|
| **약(弱)** | ×0.8 | 0수 (매 수 사용 가능) | 가벼운 기술 |
| **중(中)** | ×1.5 | 2수 | 보통 기술 |
| **강(强)** | ×2.5 | 3수 | 강력한 기술 |
| **극(極)** | ×4.0 | 5수 | 최강 기술 |

### 술 대상 선택 (자동)

```
공격 술:
  - 단일: 기력 가장 낮은 적 우선 (마무리 유도)
  - 범위: 적이 가장 밀집한 구역
  - 전체: 적 전체

회복 술:
  - 기력 비율 가장 낮은 아군 우선

가호 술:
  - 공 증가 → 아군 중 공(攻) 가장 높은 유닛
  - 수 증가 → 아군 중 기력 비율 가장 낮은 유닛
  - 신 증가 → 랜덤

저주 술:
  - 적 중 가장 위협적인(공 가장 높은) 유닛 우선
```

## 5. 기운(氣韻) 시스템

### 기운 발동 조건 유형

| 조건 유형 | 설명 | 예시 |
|----------|------|------|
| **상시** | 항상 적용 | "공(攻) +10%" |
| **기력 조건** | 기력이 특정 % 이하/이상일 때 | "기력 30% 이하 시 공(攻) 50% 증가" |
| **수 시작** | 매 수 시작 시 확률 발동 | "매 수 20% 확률로 회피" |
| **피격 시** | 피격당할 때 발동 | "피격 시 30% 확률로 반격" |
| **처치 시** | 적 처치 시 발동 | "적 처치 시 기력 10% 회복" |
| **패결 시작** | 패결 시작 시 1회 발동 | "패결 시작 시 아군 전체 공(攻) +5%" |

### 품계별 해금 및 예시

| 품계 | 해금 | 예시 (장사 기준) |
|------|------|----------------|
| 6품 (상수) | **기운 1** | 불굴 - 기력 30% 이하 시 공(攻) 50% 증가 |
| 12품 (패객) | **기운 2** | 투혼 - 적 처치 시 기력 15% 회복 |
| 18품 (진인) | **비술 해금** | (비술 시스템 참조) |
| 24품 (신투) | **초월 기운** | 불멸 - 기력 0 도달 시 1회 기력 50% 부활 |

### 기운 중첩 규칙
```
- 같은 유형의 기운: 가장 높은 수치만 적용 (중첩 불가)
- 다른 유형의 기운: 모두 적용 (중첩 가능)

예시:
  기운1 "공 +10%" + 기운2 "공 +20%" → 공 +20%만 적용 (같은 유형)
  기운1 "공 +10%" + 기운2 "피격 시 반격" → 둘 다 적용 (다른 유형)
```

## 6. 비술(秘術) 시스템

### 비술 게이지

```
┌────────────────────────────────────┐
│  비술 게이지: 0 / 100              │
│  ████████████░░░░░░░░  60%        │
│                                    │
│  충전 방법:                        │
│    - 공격 시: +10                  │
│    - 피격 시: +15                  │
│    - 적 처치 시: +20              │
│    - 매 수 자연 충전: +5           │
│                                    │
│  100 도달 시 → 다음 행동에 비술 발동│
└────────────────────────────────────┘
```

### 비술 특징
```
- 18품 이상에서 해금
- 술(術) 대신 발동 (해당 수에 술 사용 불가)
- 위력 배율: ×5.0 ~ ×8.0 (패에 따라 다름)
- 부가 효과 포함 (전체 공격, 가호, 상태이상 등)
- 발동 후 게이지 0으로 초기화
- 1회 패결에서 여러 번 발동 가능 (게이지만 다시 차면)
```

### 비술 연출
```
비술 발동 시:
  1. 슬로우 모션 (0.5초)
  2. 패 확대 컷인 (조선 수묵화 스타일)
  3. 비술명 한자 표시 (예: "秘術: 覇王斬")
  4. 특수 이펙트 + 데미지 표시
  5. 총 연출 시간: ~2초 (배속 적용)
```

## 7. 상태이상(狀態異常) 상세

### 7종 상태이상

| 상태이상 | 효과 | 지속 | 중첩 | 비고 |
|---------|------|------|------|------|
| **화독(火毒)** | 매 수 시작 시 최대 기력의 3% 감소 | 3수 | 최대 3중첩 (9%) | 데미지로 취급 (수(守) 무시) |
| **맹독(猛毒)** | 매 수 시작 시 고정 200 데미지 + 회복량 50% 감소 | 4수 | 중첩 불가, 갱신 | 회복 차단 효과 핵심 |
| **경직(硬直)** | 행동 불가 (해당 수 스킵) | 1수 | 중첩 불가, 연속 부여 불가 | 같은 대상 2수 연속 경직 면역 |
| **속박(束縛)** | 회피 불가 + 신(速) 0 | 2수 | 중첩 불가, 갱신 | 행동은 가능 |
| **혼미(昏迷)** | 행동 시 50% 확률로 아군 공격 | 2수 | 중첩 불가, 갱신 | 자해 데미지 50% |
| **둔화(鈍化)** | 신(速) 50% 감소 | 3수 | 중첩 불가, 갱신 | 행동 순서 후순위로 |
| **안개(眼蓋)** | 명중률 30% 감소 (30% 확률로 빗나감) | 3수 | 중첩 불가, 갱신 | 회피 판정과 별개 |

### 상태이상 저항

```
상태이상 저항률 = 기본 저항(0%) + 기운 보정 + 가호 보정

두목(頭目)의 기본 저항:
  - 경직: 70% 저항 (두목은 쉽게 멈추지 않음)
  - 혼미: 100% 면역 (두목은 혼란에 빠지지 않음)
  - 기타: 30% 저항
```

### 상태이상 해제
```
- 자연 해제: 지속 수 종료 시
- 술(術) 해제: 토(兎)패 일부 술에 해제 효과 포함
- 기운 해제: 특정 기운이 "패결 시작 시 상태이상 1개 해제" 등
```

## 8. 가호(加護) / 저주(詛呪) 시스템

### 가호 유형

| 가호 | 효과 | 기본 지속 | 중첩 |
|------|------|----------|------|
| **공력 증강** | 공(攻) +20~50% | 3수 | 최대치만 적용 |
| **수비 강화** | 수(守) +20~50% | 3수 | 최대치만 적용 |
| **신속 부여** | 신(速) +30% | 2수 | 최대치만 적용 |
| **기력 재생** | 매 수 기력 3~5% 회복 | 3수 | 중첩 가능 (최대 2) |
| **반격** | 피격 시 40% 확률로 반격 (공×0.5) | 2수 | 중첩 불가 |
| **보호막** | 공(攻)×2.0 만큼의 보호막 생성 | 피해 흡수 시 소멸 | 갱신 |

### 저주 유형

| 저주 | 효과 | 기본 지속 | 중첩 |
|------|------|----------|------|
| **공력 약화** | 공(攻) -20~40% | 3수 | 최대치만 적용 |
| **수비 붕괴** | 수(守) -20~40% | 3수 | 최대치만 적용 |
| **둔화** | 신(速) -50% | 3수 | (상태이상과 동일) |
| **표식** | 받는 데미지 +25% | 3수 | 중첩 불가 |
| **봉인** | 술(術) 사용 불가 (기본 공격만) | 2수 | 중첩 불가 |

### 가호/저주 규칙
```
- 같은 스탯에 가호와 저주가 동시 적용 시: 합산
  예) 공(攻) +30% 가호 + 공(攻) -20% 저주 = 순수 공(攻) +10%

- 가호/저주는 상태이상과 별개 시스템
  예) 둔화(상태이상) + 신속 부여(가호) = 양쪽 효과 동시 적용

- 해제: 지속 수 종료, 또는 특정 술로 적극 해제
```

페이즈 3 (기력 29%~0%):
  - 최종 패턴: 전체 공격 사용
  - 특수 행동: 상태이상 부여 + 자체 가호 강화
  - 분노 게이지 충전 속도 2배

## 12. 전투력 계산 공식

### 패 개별 전투력

```
패_전투력 = 기본전투력 × 품계배율 × (1 + 연마보정)

기본전투력 = (공 + 수 + 기력/10 + 신×5)
품계배율   = 1.00 ~ 13.00 (1~24품)
연마보정   = 연마단계 × 0.05 (연마 +10 = +50%)
```

### 패진 전투력 (종합)

```
패진_전투력 = Σ(패_전투력 × 8장) + 공력_보너스 + 합_보너스 + 비기_보너스

공력 보너스   = 총 공력 × 10
합 보너스     = 소합당 +500, 대합당 +1,500
비기 보너스   = 비기당 +2,000 (팔목개화 = +5,000)
```

### 전투력 표시
```
예시:
  패진 전투력: 45,230
  ├── 패 합산: 38,000
  ├── 공력: 2,300 (230 공력 × 10)
  ├── 합 보너스: 2,000 (소합 2개 × 500 + 대합 0)
  └── 비기 보너스: 2,930 (천지인 1개)
```

## 14. 데이터 모델

### 패결 데이터 구조 (참고)

```
BattleUnit {
    cardData: CardData          // 패 기본 데이터 (CardSystem 참조)

    // 패결 중 실시간 스탯
    currentHp: int              // 현재 기력
    currentAtk: int             // 현재 공(攻) (가호/저주 반영)
    currentDef: int             // 현재 수(守)
    currentSpd: int             // 현재 신(速)

    // 술 상태
    skillCooldown: int          // 술 재사용 대기 (잔여 수)
    ultimateGauge: int          // 비술 게이지 (0~100)

    // 상태이상
    statusEffects: StatusEffect[]

    // 가호/저주
    buffs: Buff[]
    debuffs: Debuff[]
}

StatusEffect {
    type: StatusType            // 화독/맹독/경직/속박/혼미/둔화/안개
    remainingTurns: int         // 잔여 수
    stacks: int                 // 중첩 수 (화독만 해당)
    source: BattleUnit          // 부여자
}

Buff / Debuff {
    type: BuffType              // 공증/수증/신증/재생/반격/보호막/표식/봉인...
    value: float                // 효과량 (%, 고정값)
    remainingTurns: int         // 잔여 수
    source: BattleUnit          // 부여자
}

BattleResult {
    isVictory: bool
    turnsUsed: int              // 소요 수
    damageDealt: int            // 총 데미지
    damageReceived: int         // 총 피해
    rewards: RewardData         // 보상 데이터
}
```
