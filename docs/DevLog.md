### 논의 내용

#### 1. 게임 방향 결정
- **장르:** 방치형 + 덱빌딩 카드 RPG
- **테마:** 조선시대 (한국 전통 문화)
- 방치형의 가벼움 + 덱빌딩의 전략적 깊이를 결합하는 하이브리드 캐주얼 방향
- 2026년 캐주얼 게임 트렌드 (방치형, 하이브리드 캐주얼)에 부합

#### 2. 게임명 선정
- 조선시대 실제 카드놀이 **투전(鬪牋)**을 핵심 소재로 채택
- 여러 후보 중 **"신투전록 (SinTujeonRok)"** 으로 확정
  - 신(神) + 투전(鬪牋) + 록(錄) = "신의 경지에 이르는 투전 기록"

#### 3. 개발 환경 세팅
- **엔진:** Unity 6 (6000.3.9f1)
- **렌더:** 2D Built-In Render Pipeline (카드게임에 적합, 가볍고 안정적)
- **프로젝트 경로:** `D:/UnityProjects/Projects/SinTujeonRok/`
- **Git/GitHub 연동 완료:** https://github.com/SmiteFLame/SinTujeonRok
- **GitHub CLI 설치 완료**

#### 5. 이슈 #1 GDD 작성 완료
- 게임 컨셉, 세계관, 스토리 정의
- 투전 팔목(8종 패) 기반 패 시스템 개요
- 엔드 콘텐츠 5종 추가 (무한 투전탑, 일일 시련, 전설의 타짜, 절기 모드, 패사 대전)
- 수익화 모델 (F2P + 인앱 + 광고)

#### 6. 이슈 #2 핵심 게임 루프 설계 완료
- 방치/수련 → 보상 → 패 수집/연마 → 패진 편성 → 패결 루프
- 방치 보상 공식 (분당 획득량 × 시간 × 보정값)
- 연마(鍊磨) (+0~+10, 성공률 체계), 각성(覺醒) (범→수→걸→신)
- 합(合) 8종 + 비기 5종 조합 시스템
- 재화 5종 설계 (은냥/옥패/수련서/패편/각성석)
- 일과(日課)/주간 콘텐츠 및 리텐션 설계

#### 7. 이슈 #3 패(牌) 시스템 설계 완료
- 64종 패 (48 일반 + 16 신물(神物))
- 24품계(品階) 시스템 (8지역 × 3품계)
- 공력(功力) 시스템 (연마 시 장착 +100, 영구 +1)
- 인연(因緣) 시스템 (지역별 3개)
- 신물 조건부 혼합 연마 해금

#### 8. 용어 사전(Terminology.md) 작성
- 조선시대 세계관에 맞는 한자어 기반 용어 체계 확립
- 주요 용어 변환: 카드→패(牌), 덱→패진(牌陣), 스킬→술(術), 강화→연마(鍊磨)
- 전투→패결(牌決), 턴→수(手), 스테이지→관문(關門), 이벤트→인연(因緣)
- 24품계 등급색 체계: 백→청→남→자→금→적→천→현
- 기존 문서(GDD, CoreGameLoop, CardSystem) 전체 용어 통일 적용
