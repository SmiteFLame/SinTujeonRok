## 2026-02-22 (Day 2) - Phase 0 완료

### 작업 내용

#### 이슈 #4 패결(牌決) 시스템 설계 ✅
- 데미지 공식 (공×위력×(1-방어율)), 방어율=수/(수+500)
- 팔목 속성 상성 (순환 구조, 유리 ×1.25 / 불리 ×0.80)
- 술(術) 4등급, 기운(氣韻) 6종 발동조건, 비술(秘術) 게이지 시스템
- 상태이상 7종 상세 (지속수, 중첩, 저항)
- 두목전 분노게이지 + 3페이즈 + 8지역 두목 예시
- 패사 대전 비동기 PvP (6등급 랭킹)
- 무한 투전탑 / 일일 시련 특수 규칙
- 전투력 계산 공식

#### 이슈 #5 방치형 보상 설계 ✅
- 방치 보상 공식 (분당×시간×관문보정×공력보정)
- 방치 vs 액티브 효율 비교 (50~70%)
- 문안/일과/주간/복귀 보상 체계
- 광고 보상 5종, 월정액 3종
- 재화 밸런스 시뮬레이션 (Day1~90)
- 보상 UI 플로우, 푸시 알림 설계

#### 이슈 #6 관문(關門)/진행 구조 설계 ✅
- 8장 × 12관문 = 96관문 (잡몹10 + 중간두목1 + 장두목1)
- 난이도 곡선 (지수적: 관문1 ×1.0 → 관문96 ×1,000)
- 관문/첫돌파/장돌파 3단계 보상
- 튜토리얼 연동 (1장 12관문에 걸쳐 시스템 순차 해금)
- 콘텐츠 해금 조건표 (패진→연마→개봉→패사대전→투전탑)
- 소탕/자동진행, 엔드콘텐츠 상세

#### 이슈 #7 UI/UX 와이어프레임 ✅
- 화면 네비게이션 흐름도 + 하단 5탭 네비게이션
- 메인 로비, 관문 선택, 패결, 결과, 패진 편성 와이어프레임
- 패 상세, 연마, 개봉, 패보, 설정 와이어프레임
- 방치 보상 팝업 UI
- UI 공통 규칙 (해상도, 터치, 색상)

#### 이슈 #8 아트 스타일 가이드 ✅
- 아트 방향: "조선 수묵화 × 모던 캐주얼"
- 색상 팔레트 (기본/품계/상태이상)
- 패 카드 디자인 템플릿 (5:7 비율, 품계별 차등)
- 캐릭터/적/NPC 디자인 방향
- UI 톤앤매너 (한지/나무/먹물)
- 이펙트/애니메이션 (수묵화 스타일)
- 무료 에셋 활용 계획 + AI 이미지 생성 활용
- 사운드 방향 (국악 퓨전)
