## 1. 게임 컨셉

### 한 줄 요약
> "조선시대 투전의 신(神)이 되어, 투전패로 세상을 평정하라"

### 핵심 키워드
`조선시대` · `투전` · `덱빌딩` · `방치형` · `카드 RPG` · `타짜`

### 컨셉 설명
신투전록은 조선시대 실존했던 카드놀이 **투전(鬪牋)**을 모티브로 한 방치형 덱빌딩 RPG입니다.
플레이어는 투전패에 깃든 신비한 힘을 다루는 **패사(牌師)**가 되어,
방치 중 자동으로 수련하고, 새로운 패(牌)를 수집하며, 패진(牌陣)을 구성하여
조선 팔도의 강적들과 패결(牌決)을 벌입니다.

### 차별점
| 기존 게임 | 신투전록 |
|----------|---------|
| 서양 판타지 카드 | **조선시대 투전** 기반 (한국 고유 소재) |
| 복잡한 조작 | **방치 + 자동 패결(牌決)** (원핸드 플레이) |
| 단순 수집 | **패진(牌陣) 전략** (합(合), 콤보) |
| 일반적 RPG 성장 | **패사(牌師) 수련** 시스템 |

## 2. 세계관 및 스토리

### 배경
조선 영조 시대, 투전은 전국을 휩쓸며 양반부터 서민까지 모두를 사로잡았다.
그러나 세상에 알려지지 않은 진실이 있었으니—
투전패에는 고대부터 전해 내려온 **신비한 기운**이 깃들어 있었고,
이 기운을 다룰 수 있는 자들을 **패사(牌師)**라 불렀다.

패사들은 투전패에 깃든 힘을 끌어내어 패결(牌決)에 사용할 수 있었다.
**인(人)패**에서는 인걸의 기상이, **마(馬)패**에서는 질풍의 속도가,
**성(星)패**에서는 천문의 지혜가 흘러나왔다.

그러나 이 힘을 악용하여 조선을 어둠에 빠뜨리려는 자들이 나타났으니,
그들은 스스로를 **흑패단(黑牌團)**이라 칭하며 팔도 곳곳에 암약하고 있었다.

### 주인공
- **이름:** 플레이어 설정 (기본명: 한수)
- **신분:** 중인 출신의 젊은 역관(譯官) 견습생
- **배경:** 역관 집안의 후예로, 우연히 고조부(장현)가 남긴 **원초 투전패**를 발견하고 패사로 각성한다.
- **목표:** 팔도를 돌며 흑패단의 음모를 저지하고, 투전의 참된 힘을 깨달아 **신의 경지(神投)**에 이르는 것.

> **참고:** 역사적으로 투전을 처음 전래한 장현(張炫)은 역관이자 장희빈의 당숙이었습니다.
> 이 실제 역사를 게임 스토리에 녹여냅니다.

### 스토리 진행 구조
```
프롤로그: 원초 투전패 발견, 패사 각성
    ↓
제1장 한양편: 한양 도성에서 기초 수련, 첫 패결(牌決)
    ↓
제2장 경기편: 흑패단의 그림자를 쫓아 경기 지역 탐색
    ↓
제3장 호서편: 충청도의 숨겨진 패사 문파 발견
    ↓
제4장 호남편: 전라도의 전설적 타짜와 패결
    ↓
제5장 영남편: 경상도의 흑패단 거점 침투
    ↓
제6장 관북편: 함경도의 극한 수련
    ↓
제7장 관서편: 평안도에서 최종 결전의 실마리
    ↓
제8장 한양 귀환: 흑패단 수장과의 최종 패결
    ↓
엔딩: 신투(神投)의 경지 달성, 엔드 콘텐츠 개방
```

### 엔드 콘텐츠 (무한 플레이)
메인 스토리 클리어는 **중반 콘텐츠**이며, 진짜 게임은 클리어 후 시작됩니다.

| 콘텐츠 | 설명 |
|--------|------|
| **무한 투전탑** | 끝없이 올라가는 탑. 층수마다 적이 강해지고, 랭킹 경쟁 |
| **일일 시련** | 매일 바뀌는 특수 룰 패결 (ex: 마패 사용 금지, 끗수 반전 등) |
| **전설의 타짜** | 스토리 이후 숨겨진 초강력 두목(頭目) 등장 (주기적 업데이트) |
| **절기(節氣) 모드** | 매 절기 새 패/스토리/인연(因緣) 추가 |
| **패사 대전** | 다른 유저의 패진(牌陣)과 비동기 PvP (실시간 아님, 방치형에 적합) |

### 주요 NPC
| 이름 | 역할 | 설명 |
|------|------|------|
| **장 노인** | 스승 | 주인공에게 투전의 기본을 가르치는 은둔한 노패사 |
| **월향** | 동료 | 기생 출신 패사, 정보 수집 담당 |
| **박거한** | 라이벌 | 양반 출신 천재 패사, 후에 동료 |
| **흑수** | 적대 | 흑패단 수장, 투전의 힘으로 조선을 지배하려 함 |
| **사또** | 의뢰인 | 각 지역에서 의뢰를 주는 관리들 |

## 3. 핵심 게임 루프

```
┌─────────────────────────────────────────────┐
│                                             │
│   [방치/수련]  ──→  [보상 획득]              │
│       ↑                  ↓                  │
│       │           [패 수집/연마(鍊磨)]        │
│       │                  ↓                  │
│       │            [패진(牌陣) 편성]          │
│       │                  ↓                  │
│       └────────  [패결(牌決)]               │
│                                             │
└─────────────────────────────────────────────┘
```

### 루프 상세
1. **방치/수련:** 오프라인 동안 자동으로 수련하여 수련치(修鍊値), 은냥(銀兩), 패편(牌片) 획득
2. **보상 획득:** 접속 시 방치 보상 수령. 시간이 길수록 보상 증가 (최대 12시간)
3. **패 수집/연마:** 투전패를 수집하고, 재료를 사용하여 연마(鍊磨)/각성(覺醒)
4. **패진 편성:** 수집한 패로 패결용 패진(牌陣)을 편성 (합(合)/콤보 고려)
5. **패결:** 편성한 패진으로 관문(關門)의 적과 자동 패결. 전략은 패진 편성에서 결정

### 세션 설계
| 세션 유형 | 시간 | 내용 |
|----------|------|------|
| 짧은 세션 | 1~3분 | 보상 수령 + 연마(鍊磨) + 다음 수련 시작 |
| 보통 세션 | 5~10분 | 위 + 패진 편성 + 관문 3~5개 돌파 |
| 긴 세션 | 15~30분 | 위 + 두목전 + 새 패 탐색 + 인연(因緣) |

## 4. 패(牌) 시스템 개요

### 팔목(八目) - 8가지 패 분류
> 실제 투전의 8가지 패 분류를 게임에 적용

| 패 이름 | 한자 | 역할 | 컨셉 |
|---------|------|------|------|
| **인(人)패** | 人 | 공(攻) | 인걸 - 직접 데미지 |
| **어(魚)패** | 魚 | 수(守) | 물고기 - 회피/방어 |
| **조(鳥)패** | 鳥 | 속공 | 새 - 빠른 연속 공격 |
| **치(雉)패** | 雉 | 범위 공격 | 꿩 - 광역 데미지 |
| **성(星)패** | 星 | 가호(加護) | 별 - 아군 강화 |
| **마(馬)패** | 馬 | 돌진 | 말 - 관통/돌파 |
| **장(獐)패** | 獐 | 저주(詛呪) | 노루 - 적 약화 |
| **토(兎)패** | 兎 | 회복 | 토끼 - 기력(氣力) 회복 |

### 패 품계(品階)
| 품계 | 색상 | 확률 | 설명 |
|------|------|------|------|
| **범(凡)** | ⬜ 백(白) | 50% | 범패(凡牌) - 일반 투전패 |
| **수(秀)** | 🟢 청(靑) | 30% | 우수한 투전패 |
| **걸(傑)** | 🔵 남(藍) | 15% | 걸출한 투전패 |
| **신(神)** | 🟡 금(金) | 5% | 신물(神物) - 신비로운 투전패 |

### 패진(牌陣) 편성 규칙
- 패진 크기: **8장** (팔목에서 착안)
- 같은 패 종류 최대 3장까지
- 합(合): 같은 종류 2장 이상 시 세트 보너스 발동
- 특수 조합: 특정 패 조합 시 **비기(秘技)** 발동

## 5. 타겟 유저

### 주 타겟
- **연령:** 20~35세
- **성별:** 남녀 공통
- **특징:**
  - 캐주얼~미드코어 모바일 게임 유저
  - 출퇴근/점심시간 등 짧은 시간에 플레이

  - 한국 역사/문화에 관심 있는 유저

- Balatro, Slay the Spire 등 덱빌딩 장르 팬

## 7. 수익화 모델

### 기본 정책
- **무료 플레이 (F2P)** + 인앱 결제 + 광고

### 수익원
| 수익원 | 설명 |
|--------|------|
| **개봉(開封)** | 옥패(玉牌)로 고품계 패 개봉 |
| **월정액 패키지** | 매일 보상 + 방치 부스터 (소액) |
| **광고 시청** | 방치 보상 2배, 무료 개봉권 획득 |
| **꾸미기** | 패 스킨, 배경 테마 (과금 압박 없음) |
| **절기(節氣) 패스** | 절기별 한정 보상 및 스토리 |

### 과금 철학
- 과금하지 않아도 모든 콘텐츠 클리어 가능
- 과금 = **시간 단축** (Pay to Skip, NOT Pay to Win)
- 최소 결제 단위: 1,000원
