# 📜 신투전록 용어 사전

> **관련 이슈:** #3 (카드 시스템 설계)
> **버전:** v0.1
> **최종 수정일:** 2026-02-21

> 신투전록은 조선시대 세계관이므로, 영어/현대 용어 대신 **한자어 기반 고유 용어**를 사용합니다.

## 3. UI 적용 예시

### 전투 화면 용어
```
기존: "Battle Start!" → "패결(牌決) 개시!"
기존: "Turn 3"        → "제3수(第三手)"
기존: "Skill: 강타"    → "술(術): 강타"
기존: "Critical Hit!"  → "급소(急所)!"
기존: "Victory!"       → "승(勝)!"
기존: "Defeat"         → "패(敗)..."
기존: "Buff Applied"   → "가호(加護) 발동"
기존: "Debuff Applied" → "저주(詛呪) 발동"
```

```
기존: "덱 편집"   → "패진(牌陣) 편성"
기존: "카드 목록" → "패보(牌譜)"
기존: "강화"      → "연마(鍊磨)"
기존: "뽑기"      → "개봉(開封)"
기존: "상점"      → "객주(客主)"
기존: "스테이지"  → "관문(關門)"
기존: "이벤트"    → "인연(因緣)"
기존: "일일 미션" → "일과(日課)"
기존: "출석"      → "문안(問安)"
```

### 카드 상세 화면
```
[패 이름] 장사(壯士)
[패 종류] 인(人)패
[품계] 상수(上手) - 6품
[등급] 🟢 청(靑)
[연마] +8

[술(術)] 강타
  - 단일 대상에게 강력한 일격
  - 위력: 150
  - 재사용: 2수

[기운(氣韻)] 불굴
  - 기력 30% 이하 시 공(攻) 50% 증가

[전투력]
  공(攻): 580
  수(守): 320
  기력:   2,400
  신(速): 85
```

## 4. 용어 사용 규칙

### 원칙
1. **게임 내 텍스트**에서는 반드시 신투전록 고유 용어를 사용
2. **괄호 표기**: 처음 등장 시 `패진(牌陣)` 형태로 한자 병기, 이후 `패진`만 사용
3. **튜토리얼**: 첫 등장 시 간단한 설명 팝업으로 용어 안내
4. **설정 옵션**: 용어 도움말 ON/OFF (툴팁으로 현대어 표시)

### 혼용 금지 목록
| ❌ 사용 금지 | ✅ 올바른 표현 |
|-------------|--------------|
| 카드 | 패(牌) |
| 덱 | 패진(牌陣) |
| 스킬 발동 | 술(術) 발동 |
| 이벤트 달성 | 인연(因緣) 달성 |
| 뽑기 | 개봉(開封) |
| 강화 +1 | 연마(鍊磨) 1단 |
| 스테이지 클리어 | 관문(關門) 돌파 |
| 레벨업 | 승품(昇品) |
| 버프/디버프 | 가호(加護)/저주(詛呪) |
